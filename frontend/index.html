Planetary Data Sonification

Mapping physical parameters of solar-system bodies to audible signals.

This project converts numerical properties of planets and moons — surface temperature, diameter, and orbital velocity — into audio signals. Originally prototyped for NASA Space Apps Challenge 2023 (Adelaide), where it was selected as a Galactic Problem Solver. Python + Streamlit for the reference implementation; this page implements the same formulas directly in the Web Audio API so you can hear each body in-browser.

§2Sonification Mapping

Each body is rendered as a single sine-wave pitch with an ADSR envelope. Pitch scales with temperature, duration with diameter, and amplitude with orbital velocity.

PITCH f = 220 × 2((T − 270) / 50) Hz DURATION d = 0.4 + 2.0 × log₁₀(D / 3000) seconds AMPLITUDE a = 0.20 + 0.60 × (V / 50) WAVEFORM sin(2πft) × ADSR envelope attack 0.02s · decay 0.10s · sustain 0.7 · release 0.3s

§3Image-to-Music Mapping

An image is divided into a grid; each cell's average luminance is mapped to the nearest note in a pentatonic scale (C, D, E, G, A). Playback proceeds left-to-right, top-to-bottom at a configurable tempo. Below: a 16×8 starfield rendered to canvas, sonified on demand.
